# Core Principle

Deadlock is not:
```
some goroutine is waiting
```

Deadlock is:

```
no goroutine can ever become runnable again
```

## Runtime View of Goroutines

Each goroutine has status:

```
_Grunnable   ready to run
_Grunning    executing now
_Gwaiting    blocked (channel/timer/etc)
_Gsyscall    in syscall
_Gdead       finished
```

### Where It Happens

In runtime scheduler code paths like:

```
schedule()
findRunnable()
checkdead()
```

The famous internal function is:

```
checkdead()
```

## Core concept

```go
func checkdead() {
    if some goroutine runnable/running:
        return

    if some timer exists:
        return

    if netpoll may wake:
        return

    if some thread in syscall may return:
        return

    if special runtime threads alive:
        return

    throw("all goroutines are asleep - deadlock!")
}
```
