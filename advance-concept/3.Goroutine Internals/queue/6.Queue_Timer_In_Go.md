# CASE 3 — time.Sleep / Timer Deep Dive

```go
time.Sleep(2 * time.Second)
```

### Runtime Flow

Eventually:

```
timeSleep()
```

### Timer Object

Simplified:

```go
type timer struct {
    when int64
    f    func(...)
    arg  any
}
```

For sleep:

```
when = now + 2s
callback wakes goroutine
```

### Steps
#### 1. Create timer

Associated with current G1.

#### 2. Insert into per-P timer heap

Heap sorted by earliest deadline.
```
top = soonest timer
```

#### 3. gopark()
```
waitReasonSleep
```

## Later Scheduler Checks Timers

When time reached:

Pop timer heap top.

Run wake callback:

```
goready(G1)
```

G1 runnable again.

### Why Heap?

Efficient next-expiration lookup:

O(1) top
O(log n) insert/remove
