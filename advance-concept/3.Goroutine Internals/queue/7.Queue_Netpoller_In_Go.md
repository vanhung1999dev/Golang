# CASE 4 — Network Read Deep Dive (epoll)

### Code
```go
conn.Read(buf)
```

#### Important

Go sockets are nonblocking.

### Step 1: Try syscall read
```
read(fd)
```

If data exists → success immediately.

If no data:

```
EAGAIN
```

### Step 2: pollDesc

Each net FD has runtime poll descriptor.

Simplified:

```go
type pollDesc struct {
    rg *g   // reader waiter
    wg *g   // writer waiter
}
```

### Step 3: Store G in pollDesc
```
pd.rg = G7
```

Means:

```
G7 waits for readable event
```

### Step 4: gopark()
```
G7 sleeping.
```

## OS Event Later

Client sends packet.

Kernel socket readable.

epoll_wait returns fd.

Runtime netpoll:

```
find pollDesc(fd)
```

If readable:

```
gp = pd.rg
pd.rg = nil
goready(gp)
```

#### Then Read Retries

G7 resumes and reads bytes.
